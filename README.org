* 專案簡介 (Overview)
:PROPERTIES:
:CUSTOM_ID: overview
:END:

本專案是一個結構化的知識地圖，旨在追蹤與分析壓力容器 (Pressure Vessel) 領域的國際規範演進、設計邏輯與產業趨勢。

透過 **Docs-as-Code** 的流程，我們將複雜的工程標準（如 ASME Section VIII Div.1/2）轉化為可視化的導航圖，協助設計人員與製造廠快速掌握關鍵技術節點。

* 核心功能 (Core Features)
:PROPERTIES:
:CUSTOM_ID: features
:END:

# - *規範演進追蹤* :: 針對 ASME 2023 至 2025 版本之變更進行深度解析。
- *適任性導航* :: 根據 Mandatory Appendix 47 建立設計人員資格證明路徑。
- *風險定價邏輯* :: 提供工程顧問服務的分級 (Level 1-3) 與風險評估框架。
# - *自動化發佈* :: 透過 Emacs Org-mode 與 CI/CD 流程自動生成靜態文件。
