1. 簡介

本文件旨在規範壓力容器建造過程中，標準管材採購的必要注意事項，確保符合 ASME Sec. VIII-1 及 Sec. II 規範。

2. 採購基本需求 (Order Requirements)

採購單 (PO) 應明確包含以下參數：

[ ] 數量
長度 (Length) 或總根數 (Quantity)。
[ ] 尺寸
標稱管徑 (NPS) 及壁厚等級 (Schedule No.)。
[ ] 標準號
例如 SA-106 / SA-312。
[ ] 法規年份
須註明可接受之 ASTM 版本年份 (ASME II-A 附錄規範)。

3. 技術要求與製程規範

3.1. 製造方式 (Manufacturing)

  • 無縫 (Seamless) : 優先選用，安全性高。
  • 焊接 (Welded) : 須注意 Joint Efficiency (E 係數) 的影響。

3.2. 熱處理 (Heat Treatment)

  • 冷拔管 (Cold Drawn) : 必須檢附熱處理記錄 (Annealed/Normalized)。

3.3. 公差與試驗 (Tolerance & Test)

  • 符合 SA-530/SA-530M 通用要求。
  • 壓扁試驗 (Flattening) : NPS 2 以上管件為強制性。

4. 材質證明 (MTR) 審核決策表   LOGIC

這是現場收料的關鍵決策點：

檢查項目 判定標準 失敗處理
爐號 (Heat No.) 須與管材鋼印 100% 一致 退貨/隔離
化學成分 符合 SA 規範之 Ladle Analysis 技術審核
機械性能 Tensile / Yield 須達標 不得使用
NDE 標記 若無水壓試驗，須標註 "NDE" 補做試驗

4.1. 特殊標記規範

  • 重量標示 : 大於 NPS 4 之無縫碳鋼管，實物須有重量紀錄。

5. 關聯法規索引

  • ASME BPVC Section VIII Division 1
  • ASME B36.10M / B36.19M : 尺寸標準。
  • ASME SA-530 : 通用技術要求。
