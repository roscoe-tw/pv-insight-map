本節為壓力容器碟形端板（Torispherical Head）設計之工程判斷節點， 涵蓋厚度計算、規範選擇、焊接效率判定與常見風險。

端板設計的困難，不在公式，而在「你選擇哪一個規範」。

你現在遇到哪個問題？

  • 不知道端板厚度怎麼算
  • JIS / ASME 不知道怎麼選
  • 焊接效率 E 不知道用多少
  • 計算結果無法通過審查
  • 軟體與手算結果不一致

設計輸入條件（Design Input）

  • 設計壓力：P
  • 設計溫度：T
  • 材料（Material）
  • 容許應力：S
  • 焊接效率：E
  • 腐蝕裕量：tc
  • 內徑：Di
  • 冠半徑：R
  • 折邊半徑：r

材料容許應力 S 必須與設計溫度對應，不能使用常溫數據。

厚度計算（Thickness Calculation）

JIS B 8265（碟形端板）

常見形式：

  • 與球殼近似
  • 需考慮 crown 與 knuckle 幾何

（此處可接 Maxima 或你既有公式系統）

ASME Section VIII Div.1

  • UG-32 (dished heads)

ASME 與 JIS 公式形式不同，但設計精神相似： → 壓力 / 應力 / 幾何 比例控制

⚠ 關鍵工程判斷：規範選擇

你不是在選公式，你是在選責任體系。

判斷流程（Decision Tree）

你使用哪個設計規範？

  • ASME
    • → 使用 ASME 公式
    • → 使用 ASME 焊接效率
  • JIS
    • → 使用 JIS 公式
    • → 使用 JIS 焊接效率
  • 混用（JIS + ASME 材料）
    • → ⚠ 必須確認：
      • 法規接受性
      • 檢驗體系
    • → 否則：
      • ❌ 不可設計

工程原則

  • 設計規範 = 檢驗規範
  • 材料來源 ≠ 規範依據
  • 不可跨規範選擇「較有利條件」

⚠ 焊接效率（Joint Efficiency, E）

ASME（Section VIII Div.1）

  • 無 RT → 0.7
  • Spot RT → 0.85
  • Full RT → 1.0

JIS（厚生勞動省告示）

  • Spot RT 可達：
    • E = 0.95
    • 抽照比例 ≥ 20%

⚠ 常見錯誤

  • 使用 JIS 設計
  • 卻套用 ASME 材料
  • 再使用 JIS 高 E（0.95）

結果：

  • 厚度低估
  • 無法通過 ASME 檢驗
  • 設計責任不成立

工程風險（Engineering Risk）

設計風險

  • 焊接效率錯誤
  • 容許應力使用錯誤
  • 腐蝕裕量遺漏

認證風險

  • 設計規範與檢驗不一致
  • 材料與規範衝突
  • 文件無法追溯

法律風險

  • 設計不符合規範
  • 發生事故責任歸屬設計者

計算錯誤是技術問題，規範錯誤是責任問題。

程式設計觀點（Software Perspective）

目前問題：

  • 材料數據需手動輸入
  • 焊接效率需人工判定
  • 規範無法自動切換

建議系統架構

  • 材料資料庫（Material DB）
  • 規範模組（Code Module）
    • ASME
    • JIS
  • 焊接效率自動對應
  • 計算書自動輸出（PDF / HTML）

工程軟體的核心不是計算，而是「避免錯用規範」。

案例（Case Study）

  • P = 1.0 MPa
  • T = 50 ℃
  • Material = SA-516M Gr.485
  • S = 138 MPa
  • E = 0.85
  • Di = 1230 mm
  • R = 1230 mm
  • r = 123 mm

討論重點

  • 為何 E 不採用 0.95？
  • 若改為 ASME 設計，厚度變化？
  • 是否符合出口（如 EU / JP）要求？
