適用條件

當設備承受內壓時，使用 UG-27 設計。

基本公式（圓筒）

t = PR / (SE - 0.6P)

變數來源

  • P → 設計壓力（Design Pressure）
  • S → Section II-D Allowable Stress

→ 材料應力來源

  • E → Joint Efficiency（UW-12）

→ 接頭效率

工程判斷重點

1. Joint Efficiency 選錯

→ 厚度會嚴重低估或高估

2. 使用錯誤應力值

→ 溫度影響未納入

3. 忽略 corrosion allowance

→ 實際壽命不足

常見錯誤案例

  • 使用 E=1，但未做 RT
  • 使用室溫 S 值於高溫設備

結論

UG-27 不是單一公式問題，而是：

「材料 + 製程 + 檢驗」的整合結果

技術支援

  • 設計是否符合 Code
  • 厚度是否合理
