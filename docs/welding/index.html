節點說明

  • 參考 ASME Sec.VIII Div.1 Table UW-12
  • 用於壓力容器殼體厚度計算
  • Joint Efficiency (\(E\)) 取決於：
    1. 焊道類型（Type 1, Type 2, …）
    2. NDT 等級（RT1, RT2, RT3, RT4）
    3. 焊接品質及可檢測性

前置條件 (Precondition)

  • 板厚 t
  • 母材材質
  • 焊道類型
  • 設計壓力 P 與設計溫度 T
  • NDT 要求（RT / UT 等級）

決策節點 (Decision Node)

  • 問題：焊接接頭效率如何確定？
  • 說明：
    • UW-12 提供 \(\eta\) 公式
    • 非破壞檢驗結果與焊接方式影響最終 joint efficiency
    • 相鄰焊道過近可能降低可檢測性 → 影響工程判斷

工程判斷分支

分支 1：全部焊道可檢測且間距合理

  • 採用 Code 建議接頭效率
  • NDT 等級符合 UW-12
  • 無需額外補償措施

分支 2：焊道間距不足或檢測受限

  • 採取補償措施：
    • 提升 NDT 等級（UT / Phased Array）
    • 分段 / Staggered 焊接
    • 預留 Inspection Window
    • PWHT 評估（板厚 > 38 mm）
  • 參考 焊道間距 Decision Node 了解詳細策略

案例 (Failure Case)

  • 板厚 20 mm
  • 焊道間距 2t
  • RT 判讀困難 → 全部焊道 gouging
  • 工期與成本增加
  • Joint efficiency 無法達到設計預期

參考文件 (Reference)

  • ASME Section VIII Div.1, 2025 Edition, Table UW-12
  • CNS 9788
